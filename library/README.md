# Shared Library Check Script
공유 라이브러리 확인 스크립트

## 스크립트 소개
* 설치 및 운영중인 프로그램 라이브러리 확인을 통해서 실행 환경 비교, 바이너리 동작 환경 검증으로 활용
* 라이브러리 파일의 Size와 Digest를 통한 버전별, 서버별 환경 비교 확인
* PKG/바이너리 업데이트에 의한 라이브러리 변화 확인
